Shiva Shankar Adari - Cyberpunk AI/ML Portfolio
A high-performance, immersive portfolio featuring dynamic weather engines and glassmorphic design.

📌 The Vision
Traditional portfolios often fail to capture the "vibe" of modern AI. This project aims to bridge that gap by using a Cyberpunk Futurism aesthetic. It treats the browser as a terminal into an AI-driven world, showcasing technical skills through interactive components rather than static text.

🚀 Advanced Features
Weather Simulation Engine: A custom-built Canvas system rendering real-time particle effects (Rain, Thunder, Snow) with an Effects Management Context.

Interactive Project Dashboard: 6+ featured AI/ML projects (from Dexterity EDTech to AIMER Society) with detailed modal views.

Experience Timeline: A vertical, animated journey of my internships and professional growth.

Glassmorphism Design System: Consistent use of translucent layers, frosted glass effects, and neon accents across all UI elements.

Mouse-Tracking Radial Glow: A responsive glow effect that follows the user's cursor, enhancing the "cyber" feel.

🛠️ Technical Specifications
Framework: React with Vite for lightning-fast HMR.

Styling: Tailwind CSS 4 (optimized for smooth 300ms transitions).

State Management: EffectsProvider for global weather and theme synchronization.

Graphics: HTML5 Canvas API for high-frame-rate particle simulations.

Type Safety: 100% TypeScript for robust, error-free deployment.

🌍 Real-World Application
This project serves as a template for Technical Branding. It demonstrates how a developer can use frontend engineering to:

Communicate Niche: The "Cyber" aesthetic immediately signals interest in advanced tech/AI.

Demonstrate Skill: Implementing Canvas and Parallax shows a level of mastery beyond standard CRUD apps.

Enhance Engagement: Interactive elements increase "Time on Page," a key metric even for recruiters.
